Сброс пароля

Пытаемся связаться с расширением…

Готово! Откройте окно расширения — можно задать новый пароль.

Если окно настроек не открылось автоматически, кликните по иконке расширения и откройте «Настройки».

Не удалось связаться с расширением.

Вы можете вставить токен вручную в окно расширения.

  1. Откройте расширение Chrome Password Lock в этом браузере.
  2. Найдите пункт «Вставить токен из письма» или кнопку сброса.
  3. Вставьте токен из поля выше и подтвердите.

Подсказка: токен действителен около 10 минут.

Нет токена в ссылке.

Откройте ссылку из письма ещё раз или запросите сброс пароля заново.

Кажется, эта страница открыта в браузере без установленного расширения.

Откройте эту ссылку в том же браузере, где установлено расширение Chrome Password Lock.
